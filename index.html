Introduction

Bienvenue sur Scrabble Wolof. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application.

Informations Collectées

Données que vous fournissez

  • Nom de joueur : Le pseudonyme que vous choisissez pour jouer (stocké localement sur votre appareil)

Données collectées automatiquement

  • Statistiques de jeu : Scores, parties jouées, victoires (stockées localement)
  • Classement mondial : Si vous participez au classement mondial, votre pseudonyme et meilleur score sont partagés de manière anonyme
  • Données d'analyse : Informations techniques anonymes pour améliorer l'application (via Firebase Analytics)

Données publicitaires

Notre application utilise Google AdMob pour afficher des publicités. AdMob peut collecter :

  • Identifiant publicitaire de l'appareil
  • Informations sur les interactions avec les publicités

Utilisation des Données

Nous utilisons vos données pour :

  • Sauvegarder votre progression de jeu
  • Afficher le classement mondial
  • Améliorer l'expérience utilisateur
  • Afficher des publicités pertinentes

Stockage et Partage

  • Données locales : Stockées sur votre appareil uniquement
  • Classement mondial : Stocké sur Firebase (Google Cloud) de manière sécurisée
  • Aucune donnée personnelle sensible n'est collectée ou stockée
Nous ne vendons pas vos données personnelles. Vos données peuvent être partagées uniquement avec Firebase (Google) et Google AdMob pour les fonctionnalités de l'application.

Vos Droits

  • Supprimer vos données : Désinstallez l'application pour supprimer toutes les données locales
  • Refuser les publicités personnalisées : Via les paramètres de votre appareil Android
  • Demander la suppression de vos données du classement mondial en nous contactant

Sécurité

Nous prenons des mesures raisonnables pour protéger vos informations :

  • Chiffrement des données en transit
  • Accès limité aux données sur nos serveurs
  • Mise à jour régulière de nos pratiques de sécurité

Enfants

Notre application est adaptée à tous les âges (PEGI 3). Nous ne collectons pas sciemment d'informations personnelles auprès d'enfants de moins de 13 ans au-delà de ce qui est nécessaire pour le fonctionnement du jeu.

Tambali

Dalal-jàmm ci Scrabble Wolof. Bataaxal bii day wax naka ñu jëlée yi nga ñu may, naka ñu ko jëfandikoo, te naka ñu koo aar.

Li ñu mën a jël

  • Sa tur : Tur bi nga tànn ngir wure (dëkk ci sa telefon)
  • Sa natt yi : Poñ yi, wure yi (dëkk ci sa telefon)
  • Dayo àdduna : Soo bokk ci dayo àdduna bi, sa tur ak sa poñ bu gën a mag dañu koy séddel

Naka ñu jëfandikoo

  • Ngir denc sa wure
  • Ngir won dayo àdduna bi
  • Ngir yokk jëf application bi
  • Ngir won fésal yi

Sa sañ-sañ

  • Mën nga far sa jëf yi : Fey application bi
  • Mën nga bañ fésal yi personalisées
  • Mën nga laaj ñu far sa jëf ci dayo bi
